
## NOVA : 노후를 바로잡다 🍀
  금융 이해력이 낮은 20대 들을 위한 “노후 준비 + 자산관리 교육” 서비스.  
  취업나이를 27로 설정하여 27세부터 60세까지 자산운용 시뮬레이션을 하고 노후에 어느정도의
  재산을 쌓을 수 있을지 측정해본다.

## 1.목표와 기능 ✈
### 1.1 목표
  - 나만의 공연 리뷰 기록 앱
  - 공연 정보를 찾아보고 계획을 세우는 캘린더 역할
  - 공연 후기를 저장해 나의 추억을 저장하는 공간
  
### 1.2 기능
  - 매년 해마다 갱신되는 뉴스
  - 매년 해마다 바뀌는 주식과 부동산 가격
  - 주식과 부동산을 통한 투자
  - 적금과 IRP를 통한 저축
  - 결혼과 출산 이벤트
